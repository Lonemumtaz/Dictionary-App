
# Lexicon — Dictionary

Lexicon is a modern, interactive dictionary web app that lets you instantly search for English word definitions, pronunciations, and example usages. Built with HTML, CSS, and JavaScript, it features a clean, responsive design and a custom audio player for pronunciation.

## Features
- Instant search for word definitions and meanings
- Pronunciation audio with custom play/pause button
- Example sentence display (if available)
- Responsive, elegant user interface
- Error handling for not found words and network issues
- Secure: prevents XSS by escaping all API/user content

## How to Use
1. Open `index.html` in your web browser.
2. Type a word into the search box.
3. Click the **Search** button or press **Enter**.
4. View the word's definition, part of speech, phonetic spelling, and example (if available).
5. Click the play button to hear the pronunciation (if audio is available).

## Project Structure
- `index.html` – Main HTML file and UI
- `style.css` – App styling and layout
- `script.js` – All JavaScript logic (API calls, rendering, audio, error handling)

## Deployment
You can deploy Lexicon using GitHub Pages, Netlify, Vercel, or any static hosting service.
