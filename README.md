# Dictionary App

A simple and interactive dictionary web application that allows users to search for word definitions, pronunciations, and examples. Built with HTML, CSS, and JavaScript.

## Features
- Search for word meanings and definitions
- Get pronunciation and example usage
- Clean and responsive user interface

## How to Use
1. Open the app in your browser (open `index.html`).
2. Enter a word in the search box.
3. Click the search button to view the definition, pronunciation, and example.

## Project Structure
- `index.html` – Main HTML file
- `style.css` – Styles for the app
- `script.js` – JavaScript logic for fetching and displaying dictionary data

## Deployment
You can deploy this app using GitHub Pages, Netlify, or Vercel for free.
